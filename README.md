# Cricket Match Predictor - Machine Learning Project

This project implements a complete machine learning system to predict cricket match outcomes using historical data. It includes a training script, a backend API, and frontend integration examples.

## Project Structure

- `ipl/`: Folder containing the dataset CSV files.
- `train_model.py`: Script to train the Random Forest model and save it.
- `app.py`: Flask backend API that serves predictions.
- `test_api_client.py`: Python script to test the API.
- `frontend_example.js`: JavaScript code to connect a frontend to the API.
- `cricket_model.pkl`: The saved trained model (generated by `train_model.py`).

## Setup Instructions

1.  **Backend Setup**:
    -   Navigate to the root folder.
    -   Install Python dependencies:
        ```bash
        pip install pandas scikit-learn flask joblib flask-cors
        ```
    -   Train the model:
        ```bash
        python train_model.py
        ```
    -   Start the Flask server:
        ```bash
        python app.py
        ```
        The server will start at `http://127.0.0.1:5000`.

2.  **Frontend Setup**:
    -   Navigate to the `ai-cricket-predictor` folder:
        ```bash
        cd ai-cricket-predictor
        ```
    -   Install Node dependencies:
        ```bash
        npm install
        ```
    -   Start the development server:
        ```bash
        npm run dev
        ```
    -   Open your browser at `http://localhost:3000`.

3.  **Usage**:
    -   On the web app, go to the Dashboard.
    -   Select Team 1, Team 2, Venue, Toss Winner, and Decision.
    -   Click "Predict Match Winner".
    -   The AI will display the predicted winner and win probability.

## Important Disclaimers

-   **Probability-Based**: This model provides a probability-based prediction based on historical data. It is not a guarantee of the future outcome.
-   **Not Betting Advice**: Cricket matches are highly unpredictable and influenced by factors not present in the data (player form, weather changes during match, etc.). This tool is for educational and entertainment purposes only. Do NOT use this for betting or financial decisions.
